
Descrição: O projeto consiste no desenvolvimento de uma solução tecnológica para o monitoramento de fluxo de clientes em redes de varejo, utilizando sensores de bloqueio de baixo custo. A proposta integra hardware e software para transformar a movimentação física dentro da loja em dados e insights digitais estruturados e estratégicos. 
O sistema utiliza sensores de bloqueio infravermelho conectados a microcontroladores Arduino para realizar a contagem de entrada, saída e circulação em pontos estratégicos da loja. Cada evento identificado é registrado com data e hora (timestamp) e transmitido em tempo real por meio de API para uma plataforma web centralizada. 
A plataforma digital possui acesso restrito por login e senha e disponibiliza um painel de controle (dashboard) interativo, com visualizações organizadas e de fácil interpretação. Entre as funcionalidades oferecidas estão mapas de calor baseados na planta baixa da loja e uma calculadora de conversão de vendas, permitindo que a gestão compreenda de forma digital e precisa o comportamento dos clientes. 
A solução tem como finalidade transformar eventos físicos em informações estratégicas para apoio à tomada de decisão.
